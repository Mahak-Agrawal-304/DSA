## Propositional Logic
Logic of propositions/statements about the world

## Propositional Symbols
They represent sentences or facts about the world

## Logic Connectives
* ~ not
* v or
* ^ and
* -> implication
* <-> biconditional

### NOT

|  P  |  ~P  |
|-----|------|
|  False  |  True  |
|  True  |  False  |
